# Student Grade Management System (Python CLI)

A simple Python command-line program to manage student grades.

## Features
- Add a student and grade
- Update student grade
- Delete a student
- View all students
- Menu-driven interface

## Technologies Used
- Python
- Dictionary data structure
- Functions
- Loops and conditions

## How It Works
The program uses a dictionary to store student names and their grades.  
Users can interact with the program through a command-line menu to perform different operations.
